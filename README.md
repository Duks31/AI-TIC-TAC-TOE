# AI-TIC-TAC-TOE

Using Minimax Algorithm, a more complex Tic-Tac-Toe game, played on the command line, can be played by Human and Normal computer or human or smart computer ( Minimax Algorithm ) or Computer and Smart Computer 

Minimax Algorithm 
- The min max algorithm in AI, popularly known as the minimax, 
  is a backtracking algorithm used in decision making, game theory and artificial intelligence (AI).
  It is used to find the optimal move for a player, assuming that the opponent is also playing optimally. 
  
- Here the teo players are called the maximizer and minimizer ( Maximizer tries to get the highest score possible, while the minimizer tries to 
  get the lowest score possible )
